# Apna College SIGMA 4.0 – Practice Repository

This repository contains my code and practice material from the **Apna College SIGMA 4.0** course. It is organized by topic and technology so each section of the course has its own folder.

## Repository Structure

- **CSS/** – CSS practice, mini projects, and layout experiments.
- **DSA/** – Data Structures and Algorithms in Java (arrays, linked lists, trees, backtracking, greedy, etc.).
- **HTML/** – Basic HTML practice files.
- **JS/** – JavaScript and DOM practice (including small browser projects).
- **EJSdir/** – EJS templating examples with a small Node/Express setup.
- **ExpressDir/** – Basic Express.js apps and REST API experiments.
- **MONGO/** and **MONGO2/** – MongoDB + Node/Express projects (models, routes, views).
- **Project/** – Larger Node/Express project from the course.
- **REST_Class/** – REST API examples and exercises.
- **SQLClass/** – SQL and Node.js integration examples.

The **WanderLust/** project is kept in a **separate repository** and is excluded from this one: [WanderLust-project](https://github.com/KanhaiyaChhaparwal24/WanderLust-project).

## Purpose

This repo is mainly for:

- Following along with the SIGMA 4.0 lectures.
- Practicing web development (HTML, CSS, JS, Node, Express, MongoDB, SQL).
- Implementing DSA problems in Java.

Feel free to browse folders topic-wise to see the progression of the course material.
